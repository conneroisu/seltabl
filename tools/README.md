# seltabl-lsp

This is a [language server](https://microsoft.github.io/language-server-protocol/) for [seltabl](https://github.com/conneroisu/seltabl).

## Installation

```sh
go intall github.com/conneroisu/seltabl-lsp
```

## Usage

```sh
seltabl-lsp --stdio
```

## Development

### Run tests

go test ./...
